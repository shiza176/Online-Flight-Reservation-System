# Online Flight Reservation System
Online Flight Booking System is a desktop application developed to simplify the process of booking and managing flight reservations.
The system provides an interactive interface for users to search flights, manage bookings, and perform reservation-related tasks efficiently.
## Features
- User-friendly interface  
- Flight booking and reservation management  
- Admin panel for managing system operations  
- User section for customer interactions  
- Organized project structure with multiple modules  
## Modules
- Admin Module  
- User Module  
## Technologies Used
- C#  
- .NET Framework   
- Visual Studio  
## Purpose
This project was developed to practice desktop application development, user interface design, and booking system functionality.
